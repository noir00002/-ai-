## 1. 遊戲概述

《不穩定鍊成論》是一套以 **高風險選擇、永久代價、不可回溯** 為核心的文字遊戲規則系統。

本系統不追求「公平的平衡」，  
而是追求 **選擇的重量**。

角色會變強，但世界會開始回應他們的存在。

適用於：
- 短～中程文字冒險
- 肉鴿式結構遊戲
- 高壓、低容錯的跑團體驗

## 2. 遊戲分工

### GM（主持者）
- 描述世界、節點與敵對存在  
- 裁定未定義規則  
- GM 裁定優先於條文細節  

### 玩家
- 僅進行角色行動與選擇  
- 不可回溯、不重置、不改寫結果  

### AI（系統輔助）
- 協助執行既定規則  
- 記錄角色狀態與永久變化  
- 不得主動補寫未定義內容

## 3. 核心設計原則

### 3.1 不可回溯
- 所有選擇一經結算即為事實  
- 角色狀態、削弱、標記為永久存在

### 3.2 層狀劇本結構
- 層級由大到小為「劇本」「層級」「節點」
- 玩家設定完角色或結算完上一個「劇本」後選擇一個「劇本」進入
- 一個「劇本」就是一局遊戲、一個故事
- 「劇本」依照難度和劇情有複數個「層級」，一個「層級」就是一個章節，各層級中至少擁有四個「節點」
- 玩家在各節點會有至少兩個選擇，以決定在節點中遇到什麼事件，分為🔴緊急作戰、🟡作戰、⚪️不期而遇、🔵安全角落



## 結語

這不是一套保護玩家的系統。  
這是一套 **要求玩家承擔後果的系統**。
